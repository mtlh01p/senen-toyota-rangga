This project is designed as an imaginary onboard PIS system for future Transjakarta buses.

## Setup
1. Ensure that Node.js and React are installed properly.
2. Clone the project into your VSCode or IDE workspace.
3. Type npm i to install all the dependencies you need.
4. Type npm run dev to start the project.

## Use
1. Select a BRT or non-BRT service.
2. Select start and end destinations.
3. Click Submit.
4. Enjoy. Press Next or Prev to move to the next/prev station.
